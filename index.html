Learn programming practically.

codeCertain is a simple platform aimed at helping users truly understand the practical implicatoins of their code, and how they can apply it in real-life situations.

Practical learning to fuel innovation.

Working with conventional teaching practices causes an influx of theoretical learning, where students don't understand the real life applications of their work.

At codeCertain our emphasis is on teaching the practical implications of programming to our users. Over the years, programming has become an essential skill to have. From scripting languages like JavaScript to data analysis with languages like Python & R, the impact that basic skills can have upon individuals is becoming more prominent.

The largest problem with conventional learning methods is that they don't allow for the development of real world skills. It's like knowing how a bicycle works, but not actually ever having ridden one. We offer practical learning by allowing our users to build applications similar to the ones they use on a day-to-day basis, and help them build their portfolios by connecting them to volunteer organisations.

Web Development

Front-end mastery course.

The front-end mastery course focuses around Web Development and using web technologies like HTML, CSS, and JS. Over the course of approximatley 50 hours of content, users learn the syntax of the languages and how they can apply it to real-life projects. Additional languages that are covered include preprocessors like Pug, Sass, and Babel.

Command Line

Enhance your CLI skills.

The Command Line learning course focuses around the GNU/Linux command line (although it is very applicable to other command lines as well) and learning the way of BASH. It covers a plethora of technologies including, but not limited to, networking interface management, package managemenent, file system navigation, and environment setups.

Git

Version control for your projects.

Passing around copies of your source files on USB, or editing live on a server are from the days of the past. The Git course focuses around learning the Git Command Line & Graphical interface to interact with version control. Technologies such as Git, Github Desktop, and Git commands are discussed. Basic command line knowledge is also provided.
